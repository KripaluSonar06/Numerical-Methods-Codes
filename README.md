# Numerical Methods (Analysis) Codes
Python code for some of the Numerical Analysis Methods which I know and taught in Numerical Methods Course @IITH Sem 3.

## Methods Implemented

### **Root-Finding Methods**

* Newton–Raphson Method
* Fixed Point Iteration
* Bisection Method
* Regula Falsi (False Position) Method

### **Differential Equation Solvers**

* Euler’s Method
* Runge–Kutta 4th Order (RK4)
* Adams–Bashforth Predictor & Adams–Moulton Corrector (PECE Method)

### **Numerical Integration**

* Trapezoidal Rule
* Simpson’s 1/3 Rule
* Simpson’s 3/8 Rule

### **Interpolation Techniques**

* Newton Forward & Backward Interpolation
* Stirling’s Interpolation
* Bessel’s Interpolation

### **Linear Algebra**

* Jacobi Iterative Method for solving **Ax = b**

### **Orthogonal Polynomials**

* Legendre Polynomials via **Rodrigues Formula**
* Legendre Polynomials via **Recurrence Relation**

##  Input Format

Whenever the script asks for a function:

* Use Python syntax
* Use `math` or `np` for mathematical functions

Example:

```
f(x) = math.sin(x) + x**2
```

For ODEs:

```
f(x,y) = x + y**2
```
